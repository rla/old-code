Solution for the 15-puzzle

Description

  • What is the 15-puzzle? [Wikipedia]

My algorithm

  • Search in the state space
  • A* as the search algorithm: special data structure as the priority queue
  • Limited search depth
  • Sum of the Manhattan distances as the state goodness value
  • Unbalanced binary tree as the storage for moves
  • Written in C, [zipped source]
