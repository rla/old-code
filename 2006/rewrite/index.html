Rewrite

Rewrite is a text rewriting system. The program is very simple. The input is read from the standard input and every occurence of term t is replaced by the contents of given file. Complex rewriting chains can be created.

Usage

:$ cat text.txt | rewrite term file.txt > rewritten.txt
:$ cat text.txt | rewritew term word > rewritten.txt

Install

:$ make
:# make install
