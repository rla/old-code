PHP laiendused

  • APD

    Advanced PHP Debugger. Antud vahend on mõeldud PHP koodi silumiseks. Võimalik on jälgida funktsioonide väljakutseid, mingi koodilõigu tööks kulunud aega jpm.

    APD paigaldamine käib PEAR-i kaudu käsuga pear install apd.

    Lingid

    • Ingliskeelne artikkel profileerimisest
    • APD Wiki
    • Silumisvahendid

  • BCMath

    Moodul töötamiseks arvudega, mis ületavad suuruselt standardseid C arvutüüpe. Kujutab arve stringidena.

    Antud laienduse olemasoluks on vajalik PHP kompileerimine võtmega --enable-bcmath.
