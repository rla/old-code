DOS

Ehk Disk Operating System. Ilma graafilise kasutajaliideseta 16-bitine operatsioonisüsteem-käsurea interpretaator, mis tavaliselt suutis vaid ühe programmiga töötada. DOS tähistab kettaseadmeid/partitsioone suurte tähtedega A, B jne. Tavaliselt on A ketas 3 1/2 floppy, B ketas 5 1/2 floppy ning C ketas esimene kõvaketas.

Allatõmbamised

  • FreeDos, vabavara DOS
  • Turbo Pascal 5.5, pascal keeles kirjutatud rakenduste arenduskeskkond.
  • QBasic, basic keeles kirjutatud rakenduste arenduskeskkond.
  • TurboC, c keeles kirjutatud rakenduste arenduskeskkond.

DOS-i käsurea interpretaatori standardkäsud

  • cd <kuhu> kaustapuus liikumise käsklus, cd .. läheb ühe kausta võrra puus üles.
  • <ketta/partitsiooni tähis>: antud ketta kasutapuu juurde liikumine. Näiteks A: läheb otse A kettale.
  • del <failinimi> faili kustutamine.
  • mkdir <uue kausta nimi> uue kausta loomine.
  • rmdir <kausta nimi> kausta kustutamine.
  • move <mida> <kuhu> failide liigutamine kaustapuus.
  • sys <ketta/partitsiooni tähis> muudab antud ketta käivitatavaks(kopeerib op. süsteemi antud kettale).

Standardkäskudes on võimalik kasutada märgendeid * ja ?, näiteks dir *.exe kuvab kõik antud kaustas olevad .exe lõpulised failinimed.

DOS-i standardrakendused

  • edit käivitab tekstiredaktori.
  • format käivitab ketaste/partitsioonide formaatimisrakenduse.
