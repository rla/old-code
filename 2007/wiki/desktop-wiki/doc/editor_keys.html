Wiki editor quick keys

Contents

  • ALT+L - create new list.
  • ALT+I - create new list item.
  • ALT+C - create level 1 heading (caption).
  • ALT+S - create level 2 heading (section).
  • ALT+P - create level 3 heading (paragraph).

Control

  • ALT+SHIFT+S - save current page.
  • ALT+SHIFT+C - compile current page into html.
