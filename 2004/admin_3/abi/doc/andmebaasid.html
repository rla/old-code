Seadistus

Failis conf.php asuvad mysql serveriga ühendamiseks vajalik aadress, kasutajanimi ja parool. Need tuleb seada enne lehekülje kasutamist.
Faili sisu:
<?php
$SYSTEEM['nimi']='sait1';       //saidi nimi
$SYSTEEM['host']='localhost';   //mysql serveri aadress
$SYSTEEM['user']='root';        //kasutajanimi
$SYSTEEM['pass']='';            //parool
$SYSTEEM['base']='admin';       //andmebaas
$SYSTEEM['vaikimisi_keel']='ee';//keel, milles kuvatakse esilehekülg.
?>
