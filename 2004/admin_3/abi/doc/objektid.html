Objektid

Objekt tähistab ühte osa koodist. Selle sisuks võib olla html või php. Täpsemalt on erinevaid tüüpe neli: html, text, css ja php. Kõik lehed koosnevad objektidest. Php-tüüpi objekti keelesüsteemi peab veebimeister ise kirjutama.

Pilt objektide nimekirjast

Objekti kasutamine php failis

Objekti saab väljastada php's funktsiooniga Objekt('objekti_nimi').
Näide
Kujundusefaili main.php sisu:

<?php
  Objekt('header');          //lehekülje päis
  Objekt($SYSTEEM['leht']);  //sisu
  Objekt('footer');          //jalus
?>
