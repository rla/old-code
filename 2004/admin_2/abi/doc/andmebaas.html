Muutuja nimi tähistab saidi nime. 
1) Tabel <nimi>_sisu:
	id char(30) - objekti nimi. primaarvõti.
	tekst text - objekti sisu.
	tyyp set(html, text, css, php) - objekti tüüp.
	pealkiri char(110) - kui tegemist lehega, siis selle lehe pealkiri.
	keel - antud objekti keel.

2) Tabel <nimi>_lehed: (kasutatakse objekti kui lehe määramiseks)
	id char(30) - objekti(lehe) nimi. primaarvõti.

3) Tabel <nimi>_keeled:
	keel char(2) - keele lühend. primaarvõti.
